# 規格文件：共用元件與設定

產出時間：2026-05-02 09:18:40 +0800

## 對應需求

- REQ-FE-03：`../01.Requirements/03.共用元件與設定.md`

## 技術規格

1. React 元件採 function component 與 TypeScript props 型別。
2. 路由採 `react-router-dom`，巢狀路由需對應 Vue Router 結構。
3. i18n 採 `i18next` 與 `react-i18next`，語系資源置於 `src/i18n/locales`。
4. 共用行為由 Vue mixin 改寫為 TypeScript helper/hook。
5. 樣式優先保留 Bootstrap class 與既有 `styles.css`，SCSS 資產列入不轉換項目。

## 檔案對應策略

- `.vue` → `.tsx`
- `.js`/`.ts` → `.ts` 或 `.tsx`，依是否輸出 React 元件判斷。
- `lang/*.json` → 僅轉換 `zh-TW`、`en`、`ja` 至目標 locale，其餘標記不轉換。
- `assets/*` → 標記不轉換，必要時直接引用或複製。

## 非功能規格

- 建置流程需透過 `npm run build` 與 `make build` 執行。
- 所有工作 shell 需使用 bash、`set -euo pipefail`，並可從自身所在目錄定位專案。
