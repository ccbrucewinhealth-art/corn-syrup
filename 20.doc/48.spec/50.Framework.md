# Framework 選擇記錄

## 原始系統 (Uptime-Kuma) Framework

### 後端 Framework
| 原始 | 功能 |
|------|------|
| Express | HTTP 伺服器 |
| Socket.IO | WebSocket 即時通訊 |
| Knex.js | SQL Query Builder |
| SQLite | 內嵌資料庫 |
| Axios | HTTP 客戶端 |
| Node-cron | 任務排程 |

### 前端 Framework
| 原始 | 功能 |
|------|------|
| Vue 3 | 前端框架 |
| Vite | 建置工具 |
| Bootstrap 5 | CSS 框架 |
| Socket.IO Client | WebSocket 客戶端 |
| Chart.js | 圖表 |
| vue-i18n | 國際化 |

## Rust 對應 Framework 選擇

### 後端 Web 框架 (Backend Web Server)
| 選擇 | 套件 | 說明 |
|------|------|------|
| 推薦 | axum | 現代異步 HTTP 框架，整合 Tower |
| 替代 | actix-web | 高效能但學習曲線較高 |
| 替代 | warp | 輕量但功能較少 |

### 前端框架 (Frontend)
| 選擇 | 套件 | 說明 |
|------|------|------|
| 推薦 | Yew | 類 Vue 3 的 WebAssembly 前端框架 |
| 替代 | Dioxus | 現代化 UI 框架 |
| 替代 | Leptos | 響應式框架 |
| 替代 | Maud | 模板引擎 (SSR) |

### WebSocket
| 選擇 | 套件 | 說明 |
|------|------|------|
| 推薦 | tokio-tungstenite | 與 Axum 整合良好 |
| 替代 | actix-web-actors | 與 Actix 整合 |

### 資料庫 (Database)
| 選擇 | 套件 | 說明 |
|------|------|------|
| 推薦 | sqlx | 異步 SQL， compile-time 檢查 |
| 替代 | rusqlite | 同步 SQLite |
| 替代 | diesel | ORM 但僅同步 |

### HTTP 客戶端 (HTTP Client)
| 選擇 | 套件 | 說明 |
|------|------|------|
| 推薦 | reqwest | 異步 HTTP 客戶端 |
| 替代 | ureq | 同步 HTTP 客戶端 |

### 任務排程 (Task Scheduler)
| 選擇 | 套件 | 說明 |
|------|------|------|
| 推薦 | cron | 時間表達式解析 |
| 替代 | tokio-cronjob | 任務排程 |

### 日誌 (Logging)
| 選擇 | 套件 | 說明 |
|------|------|------|
| 推薦 | tracing | 結構化日誌 |
| 替代 | log + env_logger | 基本日誌 |

### 密碼安全 (Security)

|------|------|------|

### 網路監控 (Network Monitoring)

## 建置流程

### 後端 Build
```bash
cargo build --release
```

### 前端 Build (使用 Trunk)
```bash
# 安裝 trunk
cargo install trunk

# 開發模式
trunk serve

# 生產建置
trunk build --release
```
