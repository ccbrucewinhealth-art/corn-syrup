# 目標目錄結構 (Target Directory Structure)

本文檔定義 corn-syrup Rust 專案的目標目錄結構，分為 backend 和 frontend-rust 兩大部分。

---

## 專案根目錄

```
corn-syrup/
├── Cargo.toml              # Rust 專案配置
├── Makefile               # 建置目標
├── backend/              # Axum 伺服器源碼
├── frontend-rust/        # Yew 前端源碼
├── dist/                # 編譯產出
└── 20.doc/              # 專案文件
```
